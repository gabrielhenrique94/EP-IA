Implementação em java, utilizando Eclipse

para sua execução devem ser passado os seguintes argumento na ordem em que aparecem:

0 - Arquivo em que se encontra os dados de treinamento. ex: "src/dados/optdigits_tra.txt"
1 - Arquivo em que se encontra os dados de teste. ex: "src/dados/optdigits_tes.txt"
2 - Número de neurônios que se deseja ter na camada escondida da MLP. ex: 5
3 - Número de neurônios que se deseja ter na camada de saída da MLP. ex: 1
4 - Valor para a taxa de aprendizado inicial (alpha). ex: 0.9
5 - Valor máximo de épocas. ex: 100
6 - Erro aceitável para a MLP. ex: 0.05
7 - Decisão se quer iniciar os pesos com 0s ou randomicos, para 0 colocar 0, para randomico colocar 1.
8 - Taxa mínima para considerar uma coluna dos arquivos sem valor informacional para a rede. (>= 0.95)
